INSTALLATION

Get Gaming in 60 Seconds

01

Build from Source

git clone https://github.com/S1b-Team/winewarden.git
cd winewarden
cargo build --release
02

Initialize Config

winewarden init
03

Launch Dashboard

winewarden monitor

Requirements: Linux kernel 5.11+, libseccomp-dev

Steam Integration: Add winewarden run -- %command% to launch options
